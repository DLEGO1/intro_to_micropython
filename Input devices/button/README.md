# Input Device:  Button.

Button closes the connection when pressed.

Attach the button to ground on one side and to an input pin on the other side.  The input pin is set to have an input pullup.  This is a setting that adds a resistor inside the microcontroller that is attached to 3.3V.  The pin is normally high, and goes to zero volts when the button is pressed.
